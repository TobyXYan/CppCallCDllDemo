0. Create a simple cpp class library for the C dll project, 
1. C Class Library is a dynamic class library
2. C Class Library need to have a Module Definition file to Export the functions that can be called from C++ app. And this file need to be configured in Property -> Linker -> Input -> Module Definition File
3. C++ App has to add the include directory which contains .h file of C dll, and add #using directory which contains C dll, and lib directory which contains C lib
4. C Project\Linker\Input need to add legacy_stdio_definitions.lib and .lib generated by C project.
5. Set Property -> C/C++ -> Precompiled Headers to be disabled before compiling C project, cause the project created by VS always enable this
6. Set Property -> C/C++ -> Advanced -> Compile As   as Compile as C Code(/TC)
